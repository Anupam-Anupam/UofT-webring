Join the UofT Webring

Are you a UofT student with a personal website? Join by following these steps:

  1. Fork the GitHub repository
  2. Edit the sites.js file to add your information:
    3. { name: "Your Name", url: "https://yourwebsite.com", year: "2X", program: "Your Program", social: { linkedin: "https://linkedin.com/in/yourprofile", github: "https://github.com/yourusername", twitter: "https://twitter.com/yourhandle", instagram: "https://instagram.com/yourprofile" // Add any other social platforms you use } }
  3. Submit a pull request
  4. Once approved, your site will appear in the webring!
